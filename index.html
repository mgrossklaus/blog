Open source software I created…

  • headman can be integrated in your projects to give you an overview over all your components and helps you to develop them encapsulated and independently from a backend. It supports a lot of JS rendering engines and validates all your components on the fly for html and accessibility violations.
  • typing-effect is a small ES module that creates a typing effect for one or multiple elements. Unlike other libraries, it does not push around other elements while typing.
  • text-truncation cuts off the text of one or multiple elements in a container with fixed height. A typical use case is a list with article teasers which all have the same height.
