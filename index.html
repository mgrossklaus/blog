Open source

miyagi
A node base component development tool for JavaScript template engines.
typing-effect
A small ES module that creates a typing effect for one or multiple elements. Unlike other libraries, it does not push around other elements while typing.
text-truncation
Cuts off the text of one or multiple elements in a container with fixed height. A typical use case is a list with article teasers which all have the same height.
