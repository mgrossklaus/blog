Open source

headman
is a development tool that helps you to develop your components encapsulated and independently from a backend. It supports many JS rendering engines and validates your components on the fly for html and accessibility violations.
typing-effect
is a small ES module that creates a typing effect for one or multiple elements. Unlike other libraries, it does not push around other elements while typing.
text-truncation
cuts off the text of one or multiple elements in a container with fixed height. A typical use case is a list with article teasers which all have the same height.
